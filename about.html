About the game

Maze project is a game developed in the C language using the SDL2 library. It is a labyrinth game where the player's objective is to find the exit. Despite not being created with popular game engines like Unity or Unreal, the project deliberately aims to create an old-fashioned game without relying on a game engine. The project has three main goals:

  • To explore the possibility of creating a game without a game engine.
  • To learn the mathematics and physics behind game development.
  • To evoke nostalgia for older games like Wolfenstein 3D, particularly in terms of graphics.
The Maze project draws inspiration from Wolfenstein 3D, a well-known video game. It shares similarities with Wolfenstein 3D in terms of the grid-based layout, uniform wall heights, and solid colored floors and ceilings. In this project, the game world is rendered by tracing a ray for each column of pixels on the screen. The vertical slice of the wall texture is selected and scaled based on the position of the ray's intersection with the walls. While the Maze project does not aim to replicate the complexity of Wolfenstein 3D, its objective is to establish the basic framework for creating a similar game environment and implementing player movement.


Raycasting

Ray-casting is a technique that converts a simplified map or floor plan into a 3D projection by tracing rays from the player's viewpoint. It gained popularity with the release of the game Wolfenstein 3D by iD Software in 1992. Ray-casting is fast because calculations are only required for each vertical line of the screen.
